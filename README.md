# Project Overview
This project focuses on the efficient integration of Rust, C, C++, and CUDA to provide seamless interoperability. The implementation leverages `libc` for improved performance.

## Features
- **Multi-language Integration**: Efficiently combines Rust, C, C++, and CUDA.

## Build & Run
Ensure you have the necessary dependencies installed before building the project.

### Using `build.sh`
A `build.sh` script is provided to automate the build process.
